## How to Use

1. Clone this repo:

```sh
git clone https://github.com/arv-anshul/ColdTurkeyBlocker-Pro.git
```

2. Run `main.py`

```sh
python main.py
```

3. Then, **specify your System Type** either `Mac` or `Windows`.
